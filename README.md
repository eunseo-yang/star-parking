# star-parking
크라우드 소싱 주차공간 리뷰 포럼

<h2> 서비스 한 줄 요약 </h2>
<p>주변 주차장에 대한 정보를 제공하고, 주차장에 대한 리뷰를 작성, 열람할 수 있는 웹

<h2>주요 기능</h2>
<p>
<ul>
  <li> 주차장 기본 정보 제공(이름, 주소, 이용 시간, 이용 요금, 주차가능 대수 등)</li>
  <li> 리뷰 작성 </li>
  <li> 별점 제공</li>
  <li> 구글 Map API 사용해 현재 위치 및 주차장 위치 표시 </li>
  <li> 회원 가입, 로그인 기능  </li>
  <li> 별점 정렬 리뷰 확인</li>
</ul>
</p>
